
# Wireframe — Transaksi/Leads Entry (Halaman CRM → Leads)

HEADER: "CRM – MARKETING"

SIDEBAR (collapsible): menu seperti Dashboard

MAIN (2 kolom)
[Form Lead]
- Nama Siswa (required)
- Asal Sekolah
- No WA (validasi format, masking)
- Email
- Channel (select: TikTok/Instagram/Meta/Organic/Referral/Event/...)
- Owner/Admin (assign)
- Status (new/contacted/follow_up/qualified/won/lost)
- Stage (prospect/follow_up/closing)
- Program diminati (multi-select)
- Catatan
- Expected Close Date
- [Simpan] [Simpan & Buat Aktivitas]

[Panel Kanan]
- Timeline Aktivitas (call/WA/email/meeting)
- Tambah Aktivitas: tipe, isi, next action, due date, reminder toggle
- Pipeline Visual (Prospect → Follow-up → Closing)
- Forecast: Probability x Price (program)

Bawah:
[Table] Leads Terkini (sortable/filterable, kolom: siswa, sekolah, channel, admin, status, updated_at, actions)
